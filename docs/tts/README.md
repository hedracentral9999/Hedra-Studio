# TTS prompts

Thu muc nay la nguon prompt cho pipeline TTS.

- `viral.md`: prompt mac dinh cho buoc style/viral.
- `11labs.md`: prompt he thong cho buoc them tag va nhan nha ElevenLabs V3.
- `<ten-prompt>.md`: prompt tuy chinh do nguoi dung tao trong app.

Khi tao prompt moi trong Hedra Studio, app se luu them file markdown theo ten prompt.
Vi du: prompt ten `Tu van Samsung DeX` se thanh `tu-van-samsung-dex.md`.

Settings van giu ban copy de tuong thich UI cu, nhung file `.md` la noi de doc/sua nhanh khi can debug.
